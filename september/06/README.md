## Завдання

**Назва класу:** Офіс

**Обовязкові поля класу:**
- Призвіще
- Ім'я
- Рік народження
- Стать
- Рік працевлаштування

**Обовязкові методи класу:**
- Введення даних
- Виведення даних
- Вивести всіх жінок які працюють не менше 2 років та мають вік від 30 років.

Запустіть Terminal та ведіть шлях до програми:

